Alumno: Miguel Arrondo

La aplicación permite agregar, modificar, consultar y borrar datos en una lista de pacientes. Incluye los datos personales, la información básica necesaria para la evaluación, fechas de ingreso o de sesión y de alta y tres valores que van cambiando en cada consulta (cantidad de sesiones, dolor y funcionalidad).

En esta versión se agregó una segunda tabla a la base de datos (Sesiones) y otra pantalla donde cada vez que se modifica la pantalla principal (en cada nueva sesión) se agrega una fila. Así, junto al número de H.C., Nombre y Apellido y fecha, se registra el número de la sesión y los valores de funcionalidad y dolor, además de la foto de la sesión.
Así se puede ver la evolución de cada paciente cronológicamente por separado, filtrando la tabla por HC o apellido, y generar un gráfico de barras para ilustrar las variaciones en función del tiempo.

La aplicación sigue el paradigma de Programación Orientada a Objetos (POO) y está estructurada en módulos independientes.
El Modelo se encarga de la creación y conexión a la base de datos Sqlite3 mediante el ORM Peewee. La Vista genera las GUIs y sus correspondientes funcionalidades. El Controlador inicia la aplicación a partir de una ventana de inicio de sesión, donde el usuario puede identificarse o registrarse si es la primera vez que ingresa. Los datos de registro (usuario y contraseña) se almacenan en una tabla de la base de datos.
Los decoradores registran la actividad del sistema, generando logs cada vez que un usuario inicia sesión y cuando se elimina un registro de la tabla Pacientes. Estos eventos se almacenan con marca de tiempo en archivos de texto (creados automáticamente si no existen). En el caso de las eliminaciones, el sistema establece una conexión cliente-servidor para almacenar en el servidor un archivo de respaldo (.txt) con los datos eliminados. El código correspondiente al servidor se incluye en una carpeta adjunta.
Mediante el patrón observador, se documentan automáticamente los nuevos registros de pacientes, generando un archivo .txt con la información correspondiente.
